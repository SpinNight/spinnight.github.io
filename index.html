EatSafe App Development Project

Team of 3

Due to COVID-19, many F&B outlets are forced to implement restrictions or completely not allow any customers to dine in. Having these implementation enforced, many F&B outlets have suffered major costs and Astons is not excluded. We were asked to create a website to help solve the business challenges faced by Astons.

This website was built by a team of 3. I am the team leader of this team, leading them to build this website with the use of python as a backend language, and html, css, jinja as backend language. Each of us had different things to do. What I have done is creating an admin page consisting of the display of list of users and CRUD of food item, and an user interface consisting of cart system, the editting of profile, CRUD of Ratings & Review System. Through what we have done, I have gotten an Outstanding Award from NYP to credit my hardwork and outstanding performance for this project.

  • Client:Astons
  • Category:F&B

Pokemon Console Game Project

Individual

This is a project that was assigned by NYP to students, with the purpose of creating a mini Pokemon Console game using C#. Using what we have learnt in C#, we were suppose to create a console game that allows us to add pokemon, view the pokemon list(pokedex), check if there are any pokemons to evolve, evolve pokemon and lastly evolve pokemon.

This console game was build by myself alone. With a busy schedule, I had to complete this project within 2 weeks despite the release date of project was rather reasonable. Therefore, in order to complete this project, one must have good time management. I had to manage my time well in order to complete this project with outstanding features. Clearly, hardwork pays off and this was the results that I had achieved.

  • Client: NYP
  • Category: Console Game/Application
